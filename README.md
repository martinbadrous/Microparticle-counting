# Microparticle Counting

## Description
Counting microparticles in fluid samples using computer vision.

## Installation
1. Clone the repo
2. Install dependencies using `pip install -r requirements.txt`

## Usage
Run `python detect_particles.py` on microscope images.

## Features
- Particle detection
- Counting
- Data export
